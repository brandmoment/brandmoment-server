🔥 День 2. Profiles
Создайте два профиля для агентного режима ассистента:

👉 Bug Fix — ассистент получает баг, сам находит причину, чинит, проверяет что не сломал остальное
👉 Research — ассистент получает вопрос по кодовой базе, сам исследует проект и выдаёт структурированный ответ

Для каждого профиля опишите:
👉 системный промпт / инструкцию профиля
👉 что ассистент ДОЛЖЕН делать (искать файлы, читать логи, запускать тесты, проверять зависимости)
👉 что ассистент НЕ ДОЛЖЕН делать (менять код в research, игнорировать тесты в bug fix)
👉 формат ответа (для bug fix — что нашёл, что починил, что проверил; для research — файлы, связи, выводы)

Проверьте каждый профиль на реальной задаче:
👉 Bug Fix — дайте реальный баг из вашего проекта, ассистент должен с первого запуска найти причину и выдать рабочий фикс
👉 Research — задайте вопрос типа "как у нас устроена авторизация" или "какие эндпоинты не покрыты тестами"
👉 если результат не рабочий с первого раза — дорабатывайте профиль и пробуйте снова

Усиление:
👉 добавьте третий профиль под вашу частую задачу (миграция, review, документация — что угодно)
👉 проверьте его по тому же принципу: один запуск — рабочий результат

Результат:
Два готовых профиля (Bug Fix + Research) с инструкциями + скриншоты/логи успешного выполнения с первого раза + описание что пришлось доработать в профилях после первой попытки

Формат:
Видео + Код
